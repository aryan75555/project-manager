# Node.js Installation - 2 Minutes

## Windows Installation (Recommended)

### Method 1: Direct Download (Easiest)

1. **Go**: https://nodejs.org
2. **Download**: "LTS" version (Green button)
3. **Open** the installer file (`.msi`)
4. **Click** "Next" → "Next" → "Next" → "Install"
5. **Restart** your computer
6. **Done!**

### Verify Installation

Open PowerShell and run:
```
node --version
npm --version
```

Should show something like:
```
v18.15.0
9.5.0
```

---

## If You Already Downloaded

Look for file on your Desktop or Downloads:
- `node-vXX.XXX.X-x64.msi` (click to install)

---

## After Installation

1. **Restart computer** (very important!)
2. **Open new PowerShell window**
3. Run: `node --version`
4. If it works → You're ready!
